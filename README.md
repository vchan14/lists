# Nuxt Template: 
A minimal template that allows use to quickly build different pages. Belows are the technologies used in this template:

- Nuxt UI
- Tailwind CSS
- Prettier

## To use this template
1. Add a folder inside the `pages` directory.
2. Create a `index.vue` file inside the folder.
3. Add this page to the list via `config/menuItems.ts`.
4. Start building your page!

## Setup

Make sure to install the dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```
